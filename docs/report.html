Spatial Radar Cross Section Attribution
Using Open-Source FDTD

How directional Poynting vector accumulation extracts spatial backscatter hot-spot maps from full-wave field simulation data — and what a UAV actually looks like on radar.

Report content is generated from report_composition.md

The full technical report is authored in docs/report_composition.md and converted to HTML for this page.

Option 1 — Pandoc (recommended):

pandoc docs/report_composition.md \
  --from markdown --to html \
  -o docs/report-content.html

Option 2 — Python markdown:

pip install markdown
python -c "
import markdown, pathlib
src = pathlib.Path('docs/report_composition.md').read_text()
html = markdown.markdown(src, extensions=['tables','fenced_code','toc'])
pathlib.Path('docs/report-content.html').write_text(html)
"

Once report-content.html exists in the docs/ folder, this page will load it automatically (see the inline script at the bottom of this file).
